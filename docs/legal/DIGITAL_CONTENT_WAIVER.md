# Digital Content Waiver Draft

This draft must be localized and reviewed before paid sale.

ClawDesk is delivered as downloadable digital software. If a user requests immediate access to the installer, license activation, updates, or Beta features after purchase, the checkout flow should clearly disclose any effect on cancellation or withdrawal rights required by the user's jurisdiction.

The desktop app should link this notice from the installer, About/legal panel, download page, and refund FAQ.
